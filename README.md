# RRS System Technology

Site institucional da **RRS System Technology**, desenvolvido para apresentar soluções, produtos e serviços de desenvolvimento de software.

Construído com **HTML, CSS e JavaScript**, com foco em identidade própria, responsividade, desempenho e conversão comercial.

## Principais funcionalidades

- Apresentação dos produtos Reobote (PDV, Cardápio Digital e Ordem de Serviços)
- Divulgação de desenvolvimento sob medida, integrações e consultoria
- Formulário de contato com validação e suporte a Formspree
- Páginas de Política de Privacidade e Exclusão de Dados
- Layout responsivo com assets otimizados (WebP + favicon)

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts (Sora + Manrope)

## Formulário de contato

O formulário envia as solicitações pelo **Formspree** para `rrs.system.technology@gmail.com`.

Endpoint configurado em `index.html`:

```html
data-formspree-endpoint="https://formspree.io/f/xpqvqapz"
```

Se o envio direto falhar, o site tenta abrir o e-mail do visitante como fallback (`mailto`).
